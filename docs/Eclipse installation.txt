Installing Eclipse for DeepskyLog development
---------------------------------------------

- Download and install java 6 from http://java.sun.com/javase/downloads/index.jsp (Select Java Runtime Environment (JRE) 6 Update xx)

- Download eclipse inclusive all packages for php development from http://www.eclipse.org/downloads/

- Unzip the file and start eclipse (choose a new directory for the workspace)


Install connector to bugzilla
-----------------------------
- Help -> Software updates -> Find and install
   Search for new features to install
     Europa Discovery Site
       Finish

       - Europa Discovery Site 
           Mylyn
             Select Mylyn Task List, Mylyn Focused UI, Mylyn Bridge : Eclipse 2.3.0.... and Mylyn Connector: Bugzilla 2.3.0....
          Next >

Install subversion client in Eclipse
------------------------------------

- Help -> Software updates -> Find and install
   Search for new features to install
    New Remote Site
       Name : Subversive plugin
       URL : http://download.eclipse.org/technology/subversive/0.7/update-site/
    New Remote Site
       Name : Subversive connectors
       URL : http://www.polarion.org/projects/subversive/download/eclipse/2.0/update-site/
    Finish
   Select Subversive SVN Team Provider (Incubation) 0.7.3...., Subversive SVN Connectors 2.0.3...., Native JavaHL 1.5 Implementation, JavaHL 1.5.0 Win32 Binaries 

  Next >

Your eclipse installation is finished now. 


Set up Eclipse for DeepskyLog Development
-----------------------------------------

a) Trunk
--------
- File -> New -> Other...
   Select SVN -> Projects From SVN

   URL : http://subversion.deepskylog.org/repos/deepskylog/
   User : ...
   Password: ...
   Select Save password

  Next >

    Select trunk
  Finish

    Check out as a project configured using the New Project Wizard
    Select 'PHP Project'
      DeespkyLog trunk
  Finish

b) Version 3.4
--------------
- File -> New -> Other...
   Select SVN -> Projects From SVN

   Use existing repository location:
     Next >

   Select tags -> 3.4
  Finish

    Check out as a project configured using the New Project Wizard
    Select 'PHP Project'
      DeepskyLog 3.4
  Finish


- the repositories are checked out now. You can always check for updates after a right mouse click on DeepskyLog 3.4 or DeepskyLog trunk and selecting Team -> Synchronize Repository, or Team -> Update


Set up Bugzilla integration
---------------------------

- Window -> Show View -> Other...
    Mylyn -> Task List
   OK

- Window -> Show View -> Other...
    Mylyn -> Task Repositories
   OK

- In the lower right window, you will see Task Repositories...
   Right click in the window, and select 'Add Task Repository'

   Select Bugzilla -> Next
   
   Server : http://www.deepskylog.org/bugs
   Label : DeepskyLog bugzilla

   Enter your bugzilla name and password (and select save password)

   - Create query -> Create a query from URL. Make the URL on the website.

- Close task repositories in the lower right window

- You can see your tasks in the Task List

- Double clicking the task will open up a form you can edit. After saving, the bugzilla entry will be adapted.
