5.1. DeepskyLog 2.1 installeren

We hebben geprobeerd om ervoor te zorgen dat het installeren van DeepskyLog 2.1 zo eenvoudig mogelijk kan gebeuren, zonder dat je een specialist bent op het gebied van systeemadministratie.

Om DeepskyLog te installeren heb je de volgende software nodig op je server :

  • php : Minimaal versie 4.3

  • Apache : Minimaal versie 2.0

  • mysql : Minimaal versie 3.2

Wanneer je zeker bent dat de benodigde software geinstalleerd is op de server waar je deepskylog wilt installeren, kan je deepskylog downloaden ( http://www.deepskylog.org/download.php ). Untar the tarball to the directory where deepskylog should be installed. tar zxvf deepskylog-2.0.tar.gz - Make a database in mysql where you want to store the data - If you don't have the permission to add a new database by yourself, your system administrator should add the database for you and give you the rights to use and adapt it.
